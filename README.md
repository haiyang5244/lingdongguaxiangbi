# 灵动卦象笔 · 易经六爻

一个轻灵雅致的 Web 端易经六爻起卦小工具，无需后端，单 HTML 文件即可运行。  
支持随机起卦、心念起卦，实时呈现本卦、综卦、错卦、互卦、交卦，并带有动爻呼吸灯及飘逸动画。

解卦可结合当前的各家大模型，一般提问格式，比如：
“xxxx卜卦得地山谦卦，动爻在第三爻（从下往上数），变卦得坤卦”

## ✨ 功能特点

- **起卦方式**  
  - 随机起卦：一键生成六爻卦象  
  - 心念起卦：输入文字，结合当前时辰通过哈希生成卦象，同一文字在不同小时可能得到不同卦象

- **卦象展示**  
  - 左侧竖排卦名与卦序  
  - 中央 SVG 绘制的六爻，阳爻为长横，阴爻为两短横，带有金色描边  
  - 右侧悬浮球展示“综、错、互、交”四关联卦，鼠标悬停（或移动端点击）可查看卦名与迷你卦象

- **交互与动画**  
  - **卦模式**：点击上卦/下卦区域，按先天八卦顺序循环切换，伴有涟漪扩散动画  
  - **爻模式**：点击任意爻线直接切换阴阳，带弹性缩放动画，手动变动后动爻自动消失  
  - 起卦时整体爻线淡入淡出，动爻出现时带有石青色呼吸灯效果

- **动爻显示**  
  起卦后会自动计算出一个动爻（基于六爻二进制值取模），动爻爻线会呈现缓慢呼吸的彩色光晕（默认石青色），手动改动爻线或上下卦后动爻消失。

- **离线可用**  
  所有依赖（Vue 3、GSAP）均已内联，保存为单个 `.html` 文件后，无论电脑或手机，断网也能完美运行。

- **移动端适配**  
  在窄屏设备上自动切换为竖排布局，关联卦悬浮卡片改为下方弹出，避免溢出。

## 🛠 技术栈

- [Vue 3](https://vuejs.org/) — 响应式数据与组件化渲染
- [GSAP](https://greensock.com/gsap/) — 爻线切换、卦象切换的流畅动画
- 原生 SVG + CSS — 六爻绘制、呼吸灯、毛玻璃效果
- 纯前端哈希 — 心念起卦算法

## 🚀 如何使用

### 在线使用（推荐）
直接访问：  
[https://你的用户名.github.io/你的仓库名/](https://你的用户名.github.io/你的仓库名/)  
（部署到 GitHub Pages 后即可使用）

### 离线使用
1. 下载本项目中的 `灵动卦象笔.html` 文件
2. 用任意现代浏览器（Chrome、Edge、Safari、Firefox）打开即可  
   无需安装任何软件，也无需网络连接。

### 操作说明
- 界面右上角小圆按钮可切换 **卦模式** / **爻模式**
- 右下角图钉按钮展开输入框与起卦按钮：
  - 输入文字后按“念”键起卦（回车也可）
  - 点击“时”键随机起卦
- 动爻（石青色呼吸灯）只出现在起卦后，手动修改卦象会自动清除


> 本工具采用“单文件”设计，所有代码、样式、数据表（64卦映射表）全部内联在 `灵动卦象笔.html` 中，方便离线保存与分享。
